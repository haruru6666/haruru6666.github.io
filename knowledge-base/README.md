# Deutsch Learning Knowledge Base（DLKB）

这不是聊天记录的备份，而是一套从 B1 持续维护到 C1 的个人德语教材。

## 目录

- `grammar/`：语法规则、结构和易错点
- `vocabulary/`：表达、主题词汇与固定搭配
- `writing/`：图表描述、论证和写作模板
- `speaking/`：陈述、讨论和口试材料
- `assets/`：图片、图表等附件
- `TEMPLATE.md`：新知识点的统一模板
- `CHANGELOG.md`：每次学习后的更新记录

## 维护原则

1. 每个知识点只维护一份；添加前先搜索。
2. 所有正文使用 Markdown，文件名使用小写英文和连字符。
3. 内容按“定义 → 用法 → 易错点 → 对比 → 例句 → 考试重点”组织。
4. 例句尽量包含中文释义，并标明适用语域或 CEFR 等级。
5. 每次修改同步更新 `CHANGELOG.md`，一次学习对应一次清晰的 Git 提交。

## 每次学习后的操作

1. 在聊天中提出：“把这个知识点加入知识库。”
2. 确定它属于哪个现有文件；没有合适文件时再新建。
3. 使用 `TEMPLATE.md` 整理内容，并检查重复和交叉链接。
4. 在 `CHANGELOG.md` 的当天日期下记录新增、修改或修正。
5. 在 VS Code 中预览 Markdown，然后提交到 GitHub。

推荐提交信息：

```text
docs(vocabulary): add darüber hinaus
```

## 完成标准

一个知识点只有在以下内容齐全时才算进入知识库：含义清楚、至少两个例句、一个易错点、必要的近义对比，以及对应的 CHANGELOG 记录。
