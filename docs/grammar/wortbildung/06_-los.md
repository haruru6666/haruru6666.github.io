# `-los`

## 核心意义

`Nomen + -los` 常表示没有或缺少某物。

| Nomen | Adjektiv | 中文 |
| --- | --- | --- |
| die Arbeit | arbeitslos | 失业的 |
| die Hoffnung | hoffnungslos | 无望的 |
| die Bedeutung | bedeutungslos | 无意义的 |
| der Schlaf | schlaflos | 失眠的 |
| der Fehler | fehlerlos | 无错误的 |
| das Ende | endlos | 无尽的 |

## Grammar

部分派生词出现连接音 `-s-`：`Arbeit → arbeitslos`。

## Comparison

- hoffnungslos ↔ hoffnungsvoll
- sinnlos ↔ sinnvoll
- respektlos ↔ respektvoll
