# `-wert`

## 核心意义

`-wert` 表示某人或某物值得进行相应的动作。

| Adjektiv | 中文 |
| --- | --- |
| sehenswert | 值得看的 |
| lesenswert | 值得读的 |
| lobenswert | 值得表扬的 |
| bemerkenswert | 值得注意的；非凡的 |
| empfehlenswert | 值得推荐的 |

## Example

> Die Ausstellung ist besonders sehenswert.

## Exam Usage

适合在评价书籍、城市、展览、措施和观点时使用，可替代较简单的 `gut` 或 `interessant`。
