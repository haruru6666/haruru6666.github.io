# `-lich`

## 核心意义

`-lich` 常表示具有某种性质、与某领域有关或类似某物。

| 基础词 | Adjektiv | 中文 |
| --- | --- | --- |
| der Freund | freundlich | 友好的 |
| das Kind | kindlich | 孩子般的 |
| der Staat | staatlich | 国家的 |
| der Beruf | beruflich | 职业上的 |
| die Wissenschaft | wissenschaftlich | 科学的 |
| die Wirtschaft | wirtschaftlich | 经济的；经济实惠的 |
| der Sommer | sommerlich | 夏日般的 |

## Comparison

- `kindlich`：孩子般的，通常中性或正面。
- `kindisch`：幼稚的，通常带贬义。

## Common Mistakes

`möglich` 已经词汇化，不宜通过现代构词公式机械分析。不能因为中文意思接近就创造 `machlich`。
