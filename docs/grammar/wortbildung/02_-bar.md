# `-bar`

## 核心意义

`Verb(stamm) + -bar` 常表示“可以被执行某动作”或“能够……”。

| Verb | Adjektiv | 中文 |
| --- | --- | --- |
| lesen | lesbar | 可读的 |
| essen | essbar | 可食用的 |
| hören | hörbar | 可听见的 |
| erreichen | erreichbar | 可到达；可联系到 |
| nutzen | nutzbar | 可利用的 |
| vermeiden | vermeidbar | 可避免的 |
| machen | machbar | 可行的 |

## Examples

- Diese Schrift ist kaum lesbar.
- Das Museum ist mit der U-Bahn gut erreichbar.
- Der Fehler wäre vermeidbar gewesen.

## 限制与例外

`-bar` 构词能力较强，但不能添加到任意动词后：

- sehen → sichtbar
- brennen → brennbar
- verstehen → verstehbar，但日常常用 `verständlich`

先用词尾推测，再通过上下文或词典确认。
