# B2/C1 考试高频构词规律

## 快速识别

| 想表达或识别的含义 | 优先联想到 |
| --- | --- |
| 可以被…… | `-bar` |
| 没有、缺少 | `-los` |
| 充满、富有 | `-voll` |
| 值得…… | `-wert` |
| 有能力或具备条件 | `-fähig` |
| 性质或领域关联 | `-lich` |
| 领域、来源或风格 | `-isch` |
| 带有某种特征 | `-ig` |
| 性质或行为倾向 | `-sam` |
| 按照……或在某方面 | `-mäßig` |

## 高频易混词

| 词 | 正确侧重点 |
| --- | --- |
| erreichbar | 能到达；能联系上 |
| erfolgreich | 成功的，不说 `erfolgbar` |
| verständlich | 易懂；可以理解 |
| verstehbar | 可理解，使用范围较窄 |
| möglich | 可能；可行，不说 `machlich` |
| sichtbar | 可见的 |
| offensichtlich | 显然的，不说 `offensichtbar` |

## 考试策略

1. 先确定基础词与目标词性。
2. 根据后缀推测大致含义，不立即自由造词。
3. 检查连接音、词干变化和词汇化形式。
4. 把派生词放回句子，检查语义和搭配是否自然。
5. 不确定时使用已经掌握的同义表达。

## 记忆单位

> 基础词 → 派生形容词 → 核心含义 → 固定搭配 → 例句

例如：

> vermeiden → vermeidbar → 可避免的 → leicht vermeidbar → Dieser Fehler ist leicht vermeidbar.

## 核心原则

构词规律首先是识别工具和记忆框架，不是无限制的自由造词公式。
