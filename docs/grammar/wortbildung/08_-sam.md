# `-sam`

## 核心意义

`-sam` 可构成表示性质、态度或行为倾向的形容词，但具体含义因词而异。

| 基础词 | Adjektiv | 中文 |
| --- | --- | --- |
| schweigen | schweigsam | 沉默寡言的 |
| achten | achtsam | 留心的；正念的 |
| sparen | sparsam | 节俭的；节省的 |
| wirken | wirksam | 有效的 |

## Common Mistakes

- 不能把 `-sam` 统一解释为“导致……”。
- `langsam` 虽以 `-sam` 结尾，但不宜作为现代自由派生的典型例子，应整体记忆。
