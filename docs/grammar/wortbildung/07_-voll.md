# `-voll`

## 核心意义

`Nomen + -voll` 常表示充满某物或富有某种性质。

| Nomen | Adjektiv | 中文 |
| --- | --- | --- |
| die Liebe | liebevoll | 充满爱意的 |
| die Hoffnung | hoffnungsvoll | 充满希望的 |
| die Wirkung | wirkungsvoll | 有效的 |
| der Sinn | sinnvoll | 有意义的 |
| der Respekt | respektvoll | 充满敬意的 |

## Comparison

`-voll` 与 `-los` 经常形成反义关系，但不是所有词都能自由配对。应将完整词作为单位记忆。
