# `-isch`

## 领域或所属

- die Politik → politisch
- die Biologie → biologisch
- die Technik → technisch

## 国家、地区或语言

- China → chinesisch
- Frankreich → französisch
- Europa → europäisch

国家派生形式并不统一：

- Deutschland → deutsch
- die Schweiz → schweizerisch / Schweizer，具体形式取决于搭配。

## 特征、风格或评价

- kindisch：幼稚的
- typisch：典型的
- praktisch：实用的
- heroisch：英雄般的
- poetisch：诗意的

## 限制

许多 `-isch` 词来自外来词或历史构词，不能只凭现代基础词自由推导。
