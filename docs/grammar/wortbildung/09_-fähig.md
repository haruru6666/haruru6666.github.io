# `-fähig`

## 核心意义

`-fähig` 表示有能力、具备条件或适合某种用途，而不是“容易……”。

| Adjektiv | 中文 |
| --- | --- |
| lernfähig | 有学习能力的 |
| aufnahmefähig | 有理解或吸收能力的 |
| leistungsfähig | 工作能力强；性能强 |
| zahlungsfähig | 有支付能力的 |
| teamfähig | 有团队合作能力的 |
| funktionsfähig | 能正常运行的 |

## Examples

- Moderne Systeme müssen anpassungs- und lernfähig sein.
- Das Gerät ist wieder vollständig funktionsfähig.

## Common Mistakes

不要把 `-fähig` 与 `-bar` 混为一谈：`-bar` 常强调可以被做，`-fähig` 强调主体具备能力或条件。
