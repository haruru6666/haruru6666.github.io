# Sich auswirken auf + Akkusativ

## Definition

> sich auf jemanden oder etwas auswirken

中文：对某人或某事产生影响。

## Grammar

固定结构：

> sich auswirken auf + Akkusativ

动词是可分动词，主句中 `aus` 位于句末：

> Der Stress wirkt sich negativ auf die Gesundheit aus.

完成时：

> Die Maßnahme hat sich positiv auf die Umwelt ausgewirkt.

## Common Patterns

- sich positiv auf etwas auswirken
- sich negativ auf etwas auswirken
- sich stark auf jemanden auswirken
- Auswirkungen auf + Akkusativ haben

## Examples

- Regelmäßige Bewegung wirkt sich positiv auf das Wohlbefinden aus.
- Hoher Zeitdruck kann sich negativ auf die Arbeitsleistung auswirken.
- Die wirtschaftliche Lage wirkt sich auf die Lebensqualität aus.

## Comparison

| Ausdruck | Struktur | Bedeutung |
| --- | --- | --- |
| sich auswirken auf | auf + Akkusativ | 对……产生影响 |
| Auswirkungen haben auf | auf + Akkusativ | 对……有影响 |
| beeinflussen | Akkusativ | 直接影响…… |

## Common Mistakes

- ✅ `auf die Gesundheit`，不是 `auf der Gesundheit`。
- 不要遗漏反身代词：`Der Stress wirkt sich ... aus.`
- 主句中必须分离：`wirkt ... aus`。

## Exam Usage

适合用于图表描述、原因结果分析以及健康、环境、工作等 B2/C1 主题。
