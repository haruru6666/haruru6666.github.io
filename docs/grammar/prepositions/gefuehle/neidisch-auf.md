# Neidisch auf + Akkusativ

## Meaning

羡慕或嫉妒某人拥有的事物。

## Grammar

> auf + Akkusativ

## Logic

情绪指向拥有优势的人或事物。

## Example

> Er ist auf den Erfolg seines Kollegen neidisch.

## Comparison

通常涉及希望自己也拥有对方的优势。
