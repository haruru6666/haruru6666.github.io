# Stolz auf + Akkusativ

## Meaning

为某人或某事感到骄傲。

## Grammar

> auf + Akkusativ

## Logic

自豪感指向某人、成果或成就。

## Example

> Meine Eltern sind auf mich stolz.

## Comparison

即使目标是事物也使用 `auf + Akkusativ`。
