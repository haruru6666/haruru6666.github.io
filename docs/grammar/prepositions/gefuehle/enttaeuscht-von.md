# Enttäuscht von + Dativ

## Meaning

对某人或某物感到失望。

## Grammar

> von + Dativ

## Logic

失望由未达到期待的人或事物引起。

## Example

> Ich bin von dem Film enttäuscht.

## Comparison

若强调具体行为，也可说 `über sein Verhalten enttäuscht sein`。
