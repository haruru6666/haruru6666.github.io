# Wütend auf + Akkusativ

## Meaning

对某人非常愤怒。

## Grammar

> auf + Akkusativ

## Logic

强烈愤怒指向某人或对象。

## Example

> Er ist auf seinen Chef wütend.

## Comparison

`wütend` 通常比 `böse` 程度更强。
