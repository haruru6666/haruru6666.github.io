# Entsetzt über + Akkusativ

## Meaning

对某事感到震惊或惊恐。

## Grammar

> über + Akkusativ

## Logic

事件或行为引起强烈震惊。

## Example

> Alle waren über die Nachricht entsetzt.

## Comparison

程度通常强于 `erstaunt über`。
