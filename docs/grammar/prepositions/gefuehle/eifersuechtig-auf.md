# Eifersüchtig auf + Akkusativ

## Meaning

嫉妒某人；吃醋。

## Grammar

> auf + Akkusativ

## Logic

嫉妒指向竞争对象。

## Example

> Sie ist auf ihre Freundin eifersüchtig.

## Comparison

不要与 `neidisch auf` 完全等同；`eifersüchtig` 常涉及关系或害怕失去某人。
