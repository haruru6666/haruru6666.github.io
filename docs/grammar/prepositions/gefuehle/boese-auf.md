# Böse auf + Akkusativ

## Meaning

生某人的气。

## Grammar

> auf + Akkusativ

## Logic

情绪指向某个对象。

## Example

> Ich bin auf dich böse.

## Comparison

与 `sich über etwas ärgern` 区分：后者强调引发愤怒的事情。
