# Beeindruckt von + Dativ

## Meaning

被……打动；对……印象深刻。

## Grammar

> von + Dativ

## Logic

印象由某人、表现或事物带来。

## Example

> Wir sind von ihrer Leistung beeindruckt.

## Comparison

不要机械使用 `auf`；这里强调感受来源。
