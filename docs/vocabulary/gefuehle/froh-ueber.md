# Froh über + Akkusativ

## Meaning

因已经发生或存在的事感到高兴。

## Grammar

> über + Akkusativ

## Logic

某个事实或消息带来喜悦。

## Example

> Ich bin über deine Nachricht froh.

## Comparison

常与 `froh auf` 混淆；标准搭配是 `froh über`。
