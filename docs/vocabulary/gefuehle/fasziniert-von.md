# Fasziniert von + Dativ

## Meaning

被……迷住。

## Grammar

> von + Dativ

## Logic

强烈兴趣由某人或某物引起。

## Example

> Sie ist von der Ausstellung fasziniert.

## Comparison

与 `interessiert an` 区分：`fasziniert von` 程度更强并强调来源。
