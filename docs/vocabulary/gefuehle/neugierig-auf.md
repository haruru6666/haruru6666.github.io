# Neugierig auf + Akkusativ

## Meaning

对即将接触的事物好奇或期待。

## Grammar

> auf + Akkusativ

## Logic

好奇心指向期待了解的目标。

## Example

> Ich bin auf das Ergebnis neugierig.

## Comparison

也可说 `neugierig auf etwas sein`，不要写成 `über`。
