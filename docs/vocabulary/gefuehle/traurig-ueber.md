# Traurig über + Akkusativ

## Meaning

因某事感到难过。

## Grammar

> über + Akkusativ

## Logic

事情或消息引起悲伤。

## Example

> Sie ist über seine Entscheidung traurig.

## Comparison

若表达为某人感到难过，需要结合具体语境说明原因。
