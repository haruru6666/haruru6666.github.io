# Seine Wut an jemandem auslassen

## Meaning

把怒气发泄在某人身上。

## Grammar

> an + Dativ

## Logic

愤怒的真正原因可能在别处，但被发泄到某人身上。

## Example

> Er lässt seine Wut an mir aus.

## Comparison

这才表示“迁怒”，`böse auf` 的语序变化不表达该含义。
