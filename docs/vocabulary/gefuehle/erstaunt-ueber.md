# Erstaunt über + Akkusativ

## Meaning

对某事感到惊讶。

## Grammar

> über + Akkusativ

## Logic

某个事实或结果引起惊讶。

## Example

> Ich bin über das Ergebnis erstaunt.

## Comparison

`über` 引出产生情绪的原因。
