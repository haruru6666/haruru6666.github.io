# Begeistert von + Dativ

## Meaning

被某人或某物深深吸引。

## Grammar

> von + Dativ

## Logic

热情或着迷感来自某个来源。

## Example

> Ich bin von der Musik begeistert.

## Comparison

`von` 引出感受来源，不是情绪目标。
