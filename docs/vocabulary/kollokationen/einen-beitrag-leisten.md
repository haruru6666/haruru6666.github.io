# Einen Beitrag leisten

## Definition

> einen Beitrag leisten

中文：作出贡献。

## Grammar

`einen Beitrag` 是第四格宾语。表达“为……作贡献”时常用：

> einen Beitrag zu + Dativ leisten

## Common Patterns

- einen wichtigen Beitrag leisten
- einen Beitrag zum Klimaschutz leisten
- einen wesentlichen Beitrag zur Lösung des Problems leisten

## Examples

- Jeder kann einen Beitrag zum Klimaschutz leisten.
- Bildung leistet einen wichtigen Beitrag zur gesellschaftlichen Entwicklung.

## Exam Usage

适合用于环境、教育、社会责任和解决方案类论证。

## Common Mistakes

`zu` 后面使用第三格：

> zu dem Klimaschutz → zum Klimaschutz  
> zu der Lösung → zur Lösung
