# Zurückhaltend

## Meaning

内敛的；克制的；不张扬的。

## Word Family

`sich zurückhalten` → `zurückhaltend` → `die Zurückhaltung`

## Common Patterns

- eher zurückhaltend sein
- zurückhaltend reagieren
- Zurückhaltung zeigen

## Examples

- Er ist anfangs eher zurückhaltend.
- Seine Zurückhaltung wirkt ruhig, aber nicht unfreundlich.

## Common Mistakes

常用名词是 `die Zurückhaltung`，不是 `die Zurückhaltigkeit`。
