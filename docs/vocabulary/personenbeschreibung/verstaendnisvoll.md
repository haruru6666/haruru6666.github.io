# Verständnisvoll

## Meaning

善解人意的；体谅他人的。

## Word Family

`verstehen` → `das Verständnis` → `verständnisvoll` → `verständlich`

## Common Patterns

- verständnisvoll reagieren
- Verständnis für + Akkusativ haben
- jemandem Verständnis entgegenbringen

## Examples

- Sie reagiert geduldig und verständnisvoll.
- Vielen Dank für Ihr Verständnis.

## Common Mistakes

常用名词是 `das Verständnis`，不是 `die Verständnisvollheit`。
