# Interessiert

## Meaning

感兴趣的；表现出兴趣的。

## Word Family

`interessieren` → `das Interesse` → `interessiert` → `interessant`

## Grammar

> an + Dativ interessiert sein

## Common Patterns

- an Sprachen interessiert sein
- Interesse an + Dativ haben
- großes Interesse zeigen

## Common Mistakes

`interessiert` 描述感到兴趣的人；`interessant` 描述引起兴趣的事物。
