# Intelligent

## Meaning

聪明的；有智慧的。

## Word Family

`intelligent` → `die Intelligenz`

## Common Patterns

- intelligent wirken
- über hohe emotionale Intelligenz verfügen

## Examples

- Er ist intelligent und lernt sehr schnell.
- Emotionale Intelligenz spielt im Berufsleben eine wichtige Rolle.
