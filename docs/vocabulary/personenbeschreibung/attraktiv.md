# Attraktiv

## Meaning

有吸引力的；有魅力的。

## Word Family

`attraktiv` → `die Attraktivität`

## Common Patterns

- attraktiv wirken
- etwas attraktiv finden
- die Attraktivität steigern

## Examples

- Sie wirkt selbstbewusst und attraktiv.
- Die Attraktivität einer Person hängt nicht nur vom Aussehen ab.
