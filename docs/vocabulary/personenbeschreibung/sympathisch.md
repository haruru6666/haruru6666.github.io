# Sympathisch

## Meaning

讨人喜欢的；给人好感的。

## Word Family

`sympathisch` → `die Sympathie` → `unsympathisch`

## Common Patterns

- sympathisch wirken
- jemandem sympathisch sein
- Sympathie für + Akkusativ empfinden

## Examples

- Sie wirkt offen und sympathisch.
- Seine ruhige Art ist mir sympathisch.

## Common Mistakes

`sympathisch` 通常表示“让人有好感”，不等同于英语 sympathetic 的“同情的”。
