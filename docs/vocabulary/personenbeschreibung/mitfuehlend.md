# Mitfühlend

## Meaning

富有同情心的；有共情能力的。

## Word Family

`mitfühlend` → `das Mitgefühl`

## Common Patterns

- mitfühlend reagieren
- Mitgefühl mit + Dativ haben
- Mitgefühl zeigen

## Examples

- Sie reagierte sehr mitfühlend auf seine Situation.
- Wir haben großes Mitgefühl mit den Betroffenen.

## Common Mistakes

常用名词是 `das Mitgefühl`，不是 `die Mitfühlung`。
