# Hilfsbereit

## Meaning

乐于助人的。

## Word Family

`helfen` → `die Hilfe` → `hilfreich` → `hilfsbereit` → `die Hilfsbereitschaft`

## Common Patterns

- hilfsbereit sein
- Hilfsbereitschaft zeigen
- für seine Hilfsbereitschaft bekannt sein

## Examples

- Meine Kollegin ist zuverlässig und hilfsbereit.
- Seine Hilfsbereitschaft wird von allen geschätzt.

## Common Mistakes

名词是 `die Hilfsbereitschaft`，不是 `die Hilfsbereitskeit`。
