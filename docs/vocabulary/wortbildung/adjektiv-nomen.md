# Adjektiv → Nomen（形容词派生名词）

## Learning Goals

- 写出常见形容词对应的抽象名词。
- 识别高频构词模式与必须单独记忆的词汇。
- 使用名词表达提升 B2/C1 写作与口语的正式程度。
- 按“形容词 → 名词 → 固定搭配 → 应用句型”记忆词族。

## Grammar / Word Formation

下列词尾变化是常见模式，但不是可以机械套用的绝对规则。学习时必须同时记住名词的冠词。

| 形容词词尾 | 常见名词词尾 | Adjektiv | Nomen |
| --- | --- | --- | --- |
| `-isch` | `-ie` | sympathisch | die Sympathie |
| `-iv` | `-ität` | attraktiv | die Attraktivität |
| `-ent` | `-enz` | intelligent | die Intelligenz |
| `-ent` | `-enz` | transparent | die Transparenz |
| `-lich` | `-keit` | freundlich | die Freundlichkeit |
| `-lich` | `-keit` | ehrlich | die Ehrlichkeit |
| `-sam` | `-keit` | aufmerksam | die Aufmerksamkeit |
| `-ig` | `-keit` | zuverlässig | die Zuverlässigkeit |

### 高频特殊形式

| Adjektiv | 正确名词 | 不推荐或错误形式 |
| --- | --- | --- |
| verständnisvoll | das Verständnis | ❌ die Verständnisvollheit |
| interessiert | das Interesse | `die Interessiertheit` 极少使用，不是这里的常用表达 |
| zurückhaltend | die Zurückhaltung | ❌ die Zurückhaltigkeit |
| mitfühlend | das Mitgefühl | ❌ die Mitfühlung |
| hilfsbereit | die Hilfsbereitschaft | ❌ die Hilfsbereitskeit |

`-bereit` 构成的形容词常与 `-bereitschaft` 名词对应：

- hilfsbereit → die Hilfsbereitschaft
- einsatzbereit → die Einsatzbereitschaft
- lernbereit → die Lernbereitschaft

## Collocations

- Verständnis für + Akkusativ haben
- Interesse an + Dativ haben
- großes Interesse an etwas zeigen
- Interesse wecken / verlieren
- Aufmerksamkeit schenken
- Sympathie für + Akkusativ empfinden
- Mitgefühl mit + Dativ haben
- Hilfsbereitschaft zeigen
- für seine Hilfsbereitschaft bekannt sein

## Writing Upgrade

名词表达往往比简单的形容词句更适合正式写作。

| 基础表达 | B2/C1 升级表达 |
| --- | --- |
| Er ist freundlich. | Seine Freundlichkeit macht einen positiven Eindruck. |
| Sie ist ehrlich. | Ihre Ehrlichkeit schafft Vertrauen. |
| Er hilft anderen gern. | Seine Hilfsbereitschaft wird von allen geschätzt. |
| Sie versteht andere gut. | Ihr Verständnis erleichtert die Zusammenarbeit. |
| Ich interessiere mich für Deutsch. | Ich habe großes Interesse an der deutschen Sprache. |

C1 表达：

> Mein Interesse an der deutschen Sprache wurde durch den Kontakt mit deutschsprachigen Freunden geweckt.

## Speaking

- Was ich an ihr besonders schätze, ist ihre Aufrichtigkeit.
- Seine Hilfsbereitschaft macht ihn zu einem angenehmen Kollegen.
- Ihre Zurückhaltung wirkt zunächst distanziert, ist aber keineswegs unfreundlich.
- Er zeigt viel Mitgefühl für Menschen in schwierigen Situationen.

## Exam Usage

- B2/C1 人物描述。
- 讨论个人品质与职场能力。
- 正式邮件与投诉回复。
- 演讲结尾：`Vielen Dank für Ihre Aufmerksamkeit.`
- 表达理解：`Vielen Dank für Ihr Verständnis.`

## Common Mistakes

- 不要只根据形容词词尾猜名词；先检查德语中是否已有常用词，如 `das Verständnis`。
- 名词必须大写，并与冠词一起记忆。
- `Interesse` 常与 `an + Dativ` 搭配：`Interesse an der deutschen Sprache`。
- `Sympathie` 表示对某人的好感时常用 `für + Akkusativ`。
- `Mitgefühl` 与对象搭配时使用 `mit + Dativ`。
- 把形容词当名词使用，如 `der Arbeitslose`，与本页的“派生抽象名词”不是同一现象。

## Summary

常见模式：

- `-isch → -ie`
- `-iv → -ität`
- `-ent → -enz`
- `-lich / -sam / -ig → -keit`
- `-bereit → -bereitschaft`

重点记忆：`das Verständnis`、`das Interesse`、`das Mitgefühl`、`die Zurückhaltung`、`die Hilfsbereitschaft`。

> Adjektiv → Nomen → Kollokation → Schreib-/Sprechsatz
